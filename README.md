# Pacman-Exercise
This is a simple PacMan exercise used to demonstrate how to dynamically display data on a web page using javascript
How to Run : you should be able to drag and drop the index.html file onto the browser
Future Improvements: Adding dots for pacman to eat or you could make a game where 2 pacmen try to eat each other
